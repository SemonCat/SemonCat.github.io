最近開始使用Sublime Text 2來編寫CODE，

VMware安裝Mac OS教學

[][1]前幾天在玩MacOS，

安裝的時候遇到了一些問題，

在這邊記錄下來供日後參考。

———————————————————————————————-轉貼自OKFUN論壇

[使用說明]：

  1. 下載後解壓縮，會有 3 個資料夾
    Mac OS X Lion –> Mac OS X Lion 10.7.4 VMware 主映像檔
    VMware Unlocker - Hardware Virtualization Bypasser –> 使 VMWare 繞過對硬體虛擬化技術的檢查步驟
    VMware Unlocker - Mac OS X Guest –> 主要解除 VMware 對 Mac OS 的限制
  2. 安裝 VMware Workstation 8 (建議使用，但本文不提供)
  3. 進入「VMware Unlocker - Mac OS X Guest」資料夾，找到您使用的 VMware 版本，執行「install.cmd」解鎖
    (VMware Workstation 8.x 路徑：\VMware Unlocker - Mac OS X Guest\VMware 8.x Series\VMware Workstation Unlocker - Windows)
  4. 解鎖完成後，執行 VMware，點選 [Open a Virtual Machine] 按鈕，進入「Mac OS X Lion」資料夾載入映像檔 ( 不知道的點我看圖 )
  5. 啟動後 Mac OS X Lion 的虛擬機器後稍等一下，進入系統就成功了。
    PS: 預設帳號沒有設定密碼。

[把系統換成繁體中文]：

  1. 啟動 Mac OS X Lion 系統。
  2. 點一下左上角的蘋果圖示，點一下 [System Preferences…]。
  3. 在出現的「System Preferences」視窗中，點一下 [Languagw & Text]。
  4. 找到「繁體中文」並且直接用滑鼠拖到最上面 (或是先選好繁體中文，用鍵盤 Alt + 方向鍵 也可以移到最上面)。
  5. 改好之後，再點一下左上角的蘋果圖示，選 Log Out (登出) 或是 Restart… (重新啟動)，就可以看到系統變成繁體中文了。

[安裝修改版 VMWare Tools 以便 MAC OS X 變更解析度]：

  1. 先把 MAC OS X 做一份快照以防萬一 (VMWare 選單上的 [VM] -> [Snapshot] -> [Take Snapshot…]，按下 [Take Snapshot] 按鈕就會製作一份快照了。)
  2. 將 darwin.iso 檔案掛載到 VMWare 的 MAC OS X 的 CD/DVD 光碟機上。darwin.iso 檔案位於「VMware Unlocker – Mac OS X Guest\VMware 8.x Series\Tools」
  3. 進入 MAC OS X 的系統，等到虛擬光碟機的 VMWare Tools 掛載後，執行「Install VMware Tools 」。(預設使用者沒有密碼)
  4. 安裝後必須重新啟動 MAC OS X，重新啟動之後就可以在「顯示」中設定更多的解析度了。

[VMWare 卡在灰色蘋果開機畫面] (不一定所有人適用)：

Q: 請問有執行過 VMware Unlocker - Mac OS X Guest 資料夾中的修正程式 (解除限制) 嗎?
A: 沒有的話請根據您用的 VMWare 版本執行修正 (解除限制)。
Q: 執行過還是一樣?
A: 請將您的電腦重新開機，進入 BIOS，找到「Virtualization Technology (虛擬化技術)」，並設定為 [Enabled]，再試著啟動一次 MAC OS X。
Q: BIOS 中沒有「Virtualization Technology (虛擬化技術)」選項，或是設定啟動後結果還是一樣?
A: 請下載文章附件 vmware-vmx-patch-8.0.3.txt 的檔案，並請安裝 VMware Workstation 8.0.3 才能正常修正

WP簡訊備份

SMSBackup

WinPhone的備份機制非常差，送修的機器完全無法保留簡訊的資料，所幸對岸有高手發現到SMS其實有簡訊備份機制，於是開發出了備份軟體，寫了一下教學以防自己忘記。

以下是備份過程：

  1. 先取得WP7 Root Tools軟體。
  2. 將SMSBackup設定為信任。
  3. 使用SMSBackup，將簡訊備份。
  4. 使用可以進入WP7資料夾的軟體，將備份的store.vol、PimIndex.vol取出。

以下是還原過程：

  1. 拿到全新手機或者重灌手機以後，先把所有帳號同步都設定完畢，並且將資料同步完成。
  2. 打開SMSBackup，然後再使用WP7資料管理程式，將store.vol、PimIndex.vol放回去。
  3. 還原，然後重新同步。

如果遇到問題，請先重置，重置完先還原store.vol、PimIndex.vol，再重新同步。

EclipseWeb環境建置

最近無聊跑去玩Linux，結果發現Linux裝Apache有夠麻煩，在這邊分享一下過程，以免大家跟我一樣走了許多冤枉路。

今天就先寫Linux安裝Apache的過程，其他的有空在慢慢補上。

1.首先就是到Apache官網下載Httpd。

2.解壓縮，打開終端機，切換到自己壓縮的目錄下，輸入./configure，網路上的教學這一步驟都只有寫這個樣子，實際上還要裝許多其他Runtime，如果這一步驟順利通過沒有跳錯誤，就直接make -> make install就完成安裝了，如果跟我一樣跳錯誤的話， 應該會長這樣

checking for APR… no
configure: error: APR not found . Please read the documentation.

3.這代表沒有安裝Apache Portable Runtime，到網頁以後下載APR跟APR-util，一樣都解壓縮。

4.終端機目錄切換到APR的資料夾，輸入./configure，會產生makefile，然後在輸入make，
最後輸入make install，安裝就完成了。

5.接下來安裝APR-util，切換到APR-util目錄下，輸入

./configure –prefix=/usr/local/apr-util –with-apr=/usr/local/apr

產生makefile，然後輸入make，最後輸入make install，安裝完成。

6.接下來要安裝PCRE，一樣的方法就不再贅述了，必須注意的是，PCRE必須安裝gcc-g++。

7.再來應該可以安裝Apache了，終端機目錄指定到Httpd下，輸入

./configure –with-apr=/usr/local/apr/ –with-apr-util=/usr/local/apr-util/
–with-pcre=/usr/local/pcre

應該就可以順利安裝了。

==========暫時寫到這，完整教學後補。
